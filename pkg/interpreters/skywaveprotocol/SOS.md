6. Key Points for Valid SOS
✅ Event Code 1 = SOS Active (Input 1 Active)
✅ Event Code 9 = SOS Inactive (Input 1 Inactive)
✅ I/O Status 0x0300 = Input 1 ON + Input 2 ON (SOS pressed with ignition on)
✅ I/O Status 0x0200 = Input 1 OFF + Input 2 ON (SOS released with ignition on)
✅ Checksum must be calculated correctly
✅ IMEI must match a registered device
